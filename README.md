# 交换机日志分析工具 v1.0

一款用于批量分析交换机日志文件的桌面工具，支持自定义关键词匹配、上下文提取、结果导出等功能。

## 功能特性

- **批量文件分析**：递归扫描指定文件夹下所有 `.txt` 日志文件
- **自定义关键词管理**：
  - 支持添加、编辑、删除关键词
  - 可设置关键词的上下文提取范围（向前/向后字符数）
  - 支持关键词的导入/导出（`.kw` 格式）
  - 支持全选/取消全选关键词
- **智能内容提取**：自动提取关键词后的内容，不包含关键词本身
- **多编码支持**：自动检测文件编码，避免乱码问题
- **多视图展示**：
  - 分析结果表格：展示文件名、描述、提取信息、匹配状态
  - 详细信息视图：按文件查看完整提取结果
  - 统计摘要：展示文件数、关键词匹配统计
- **Excel 导出**：
  - 生成格式化的 Excel 报告（分析结果表 + 统计信息表）
  - 自动着色：成功匹配绿色，失败/未找到红色
  - 自动调整列宽，添加边框样式

## 运行环境

- Python 3.6 或更高版本
- 依赖库：`openpyxl`、`chardet`

## 安装与运行

### 1. 安装依赖

```bash
pip install openpyxl chardet
```

### 2. 运行程序

```bash
python 交换机日志分析工具.py
```

## 使用说明

### 界面布局

- **工具栏**：选择日志文件夹、开始分析、导出 Excel、清空结果
- **左侧面板**：自定义关键词管理（列表、编辑、导入导出）
- **右侧面板**：分析结果展示（表格、详细信息、统计摘要）
- **底部状态栏**：显示进度和状态

### 操作流程

1. **选择日志文件夹**：点击“浏览...”选择包含 `.txt` 日志文件的目录
2. **管理关键词**：
   - 使用预设的示例关键词，或自行添加/修改
   - 设置每个关键词的“向前提取”和“向后提取”字符数（控制提取范围）
   - 可选中多个关键词（Ctrl 多选），全选/取消全选
3. **开始分析**：点击“开始分析”，程序将扫描所有日志文件并匹配关键词
4. **查看结果**：
   - 在“分析结果”表格中查看匹配情况
   - 点击任意行可在“详细信息”中查看完整提取内容
   - 在“统计摘要”中查看匹配统计
5. **导出 Excel**：点击“导出 Excel”生成报告文件，可选择是否立即打开

### 关键词提取说明

- 提取时**不包含关键词本身**，只提取关键词后的内容
- 提取范围 = 关键词后指定字符数（由“向后提取”控制）
- 如果提取内容为空，会自动尝试扩展到 200 字符
- 提取内容长度限制为 200 字符，超出部分显示 `...`

### 关键词导入/导出

- **导出**：将当前关键词列表保存为 `.kw` 文件（使用 pickle 序列化）
- **导入**：从 `.kw` 文件加载关键词列表（会替换当前列表）

## Excel 报告说明

生成的 Excel 文件包含两个工作表：

1. **分析结果**：
   - 表头：序号、文件名 + 各关键词描述
   - 匹配成功：绿色背景
   - 匹配失败/未找到：红色背景

2. **统计信息**：
   - 文件总数
   - 关键词数量
   - 每个关键词的匹配统计（匹配数/总文件数）

## 注意事项

- 仅支持 `.txt` 格式的日志文件
- 大文件分析时程序可能暂时无响应，请耐心等待（分析在后台线程执行）
- 导出 Excel 前请确保已执行过分析
